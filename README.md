# Replication package: The role of interdisciplinary funding in supporting impactful interdisciplinary research

This repository provides the replication code, anonymized data, and supporting outputs for:

> Bingyi Wu, Tianjun Wu, Yanmeng Xing, Xinzhi Xie, and Chao Min. "The role of interdisciplinary
> funding in supporting impactful interdisciplinary research." *Journal of Informetrics* 20(3)
> (2026): 101844. [https://doi.org/10.1016/j.joi.2026.101844](https://doi.org/10.1016/j.joi.2026.101844)

The replication package and its documentation were organized and validated with **Codex GPT-5.6 Sol**.

## Data

The funding-project information in the shared data has been anonymized. The repository retains only
the data required to reproduce the reported analyses.

## Related model

The related deep learning model for funding subject-field tag classification is available at
[yusu2022/funding_subject_field_tag_classification](https://github.com/yusu2022/funding_subject_field_tag_classification).
